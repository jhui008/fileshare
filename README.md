# Fileshare
a simple python file upload/download web server

# Change port

`filesserver.py` : chang the `port` as you like

```py
if __name__ == "__main__":
    # server start up
    app.run(host='0.0.0.0', port=8080, debug=True)
```

# Start Server

Dependency: flask

```
python filesserver.py
```
